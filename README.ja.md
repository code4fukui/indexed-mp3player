# indexed mp3player

MP3ファイルをブラウザ上で再生できるシンプルなmp3プレーヤーアプリです。IndexedDBを使用してMP3ファイルを保存・管理し、ドラッグ&ドロップでファイルを追加できます。

## デモ
https://code4fukui.github.io/indexed-mp3player/

## 機能
- MP3ファイルのドラッグ&ドロップによる追加
- 保存したMP3ファイルの再生
- 再生中の曲の表示
- 再生履歴の保存

## 使い方
1. ブラウザ上でアプリを開きます。
2. MP3ファイルをドラッグ&ドロップしてアプリに追加します。
3. 追加したファイルをクリックすると再生されます。
4. 再生中の曲は表示が変わり、再生が終了すると自動で次の曲に切り替わります。
5. 「CLEAR」ボタンを押すと、保存された全てのMP3ファイルが削除されます。
